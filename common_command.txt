cd : 改变当前目录位置
       cd [无参数] ：返回主目录(HOME目录)
       cd [~] ：返回主目录(HOME目录)
       cd [-] ：返回最近一次访问目录
       cd [..] ：返回上一级目录
       cd [子目录名]：进入当前目录的子目录
       cd [绝对路径]：从当前目录转到别的目录
ls ：显示目录内容
       -a : 显示全部文件，包括隐藏文件
       -l  :  列举目录详细内容，包括权限、大小、创建日期等
       -F : 列举文件类型  *为可执行文件
       -r : 从后向前列举目录中的文件
       -R :  递归列举当前目录及所有子目录文件
       -S :  按文件大小从大到小排列文件
pwd : 查看当前完整路径
mkdir :  建立新目录
       -m : 指定新建目录访问权限
rmdir：删除目录
mv：移动或修改目录及文件名称  不加参数会直接覆盖同名文件
cp：复制文件或目录
rm：删除文件或目录
       -f : 强行删除，不给出任何提示
       -i : 删除前给出提示
       -r : 删除指定目录
       -R : 递归删除目录及子目录
cat：显示文件内容
grep: 查找文件内容
